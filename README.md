# Portfolio Architecture

This portfolio demonstrates a **3-stage data engineering pipeline** built around real-world traffic data.

The goal is to show how raw operational signals can move through a full data stack:

```
Video Scene Data
      ↓
Computer Vision Event Detection
      ↓
Raw Event Ingestion (Azure)
      ↓
Cloud Data Lake Storage
      ↓
Data Transformation & Analytics
      ↓
Dashboards & Insights
```

Each repository represents one stage of the system.

# Featured Projects

## 1️ Real-Time Vision Event Generator  
**Computer Vision → Structured Event Data**

This project processes **driver point-of-view traffic video** and converts object detections into **structured JSON event streams**.

Instead of simply detecting objects, the system produces **machine-readable event records** that can feed downstream data pipelines.

**Key Concepts**

- Computer vision with YOLOv8
- Structured event generation
- Video processing pipelines
- Data engineering event schema design

**Tech**

Python • OpenCV • YOLOv8 • JSON Event Streams
